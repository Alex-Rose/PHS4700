# PHS4700 - Physique pour multimédia

## Devoir 1

### Comment exécuter la simulation
Dans Matlab, exécuter le fichier *main.m*. Les résultats s'afficheront 
dans la console. 

Pour changer entre le patineur avec bras le long du corps et avec le bras
gauche à l'horizontale, changer la ligne 5 du fichier *main.m*. Inscrire :

* Declarations1 pour les bras le long du corps
* Declarations2 pour le bras gauche à l'horizontale

Pour activer la rotation de -10deg du patineur, mettre valeur de *incline* à *true*
dans *main.m* à la ligne 9
